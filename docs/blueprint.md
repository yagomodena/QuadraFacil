# **App Name**: QuadraFacil

## Core Features:

- Proprietário Cadastro: Permitir que proprietários de quadras se cadastrem na plataforma, fornecendo informações como e-mail, senha, nome do estabelecimento, quantidade de quadras, cidade, telefone e tipo de esporte das quadras.
- Proprietário Dashboard: Oferecer um painel de controle para os proprietários, com visão geral das quadras, horários ocupados, receitas, reservas pendentes, agenda visual e funcionalidades de configuração de horários, preços, histórico financeiro e gerenciamento de reservas.
- Cliente Front-end: Permitir que os clientes acessem um link exclusivo do estabelecimento, visualizem a agenda da quadra, selecionem um horário disponível, escolham entre pagamento integral ou dividido, realizem o pagamento online (Pix, cartão) e recebam confirmação automática após o pagamento.
- Divisão de Pagamento: Habilitar a divisão de pagamento entre os membros de um grupo. O sistema deve gerar links individuais para cada pessoa pagar a sua parte, e o horário só é confirmado quando todos os membros pagarem. A divisão do pagamento está disponível apenas nos planos mensais.
- Notificações Automáticas: Enviar notificações automáticas via WhatsApp ou e-mail sobre reservas e pagamentos pendentes.
- Gateway de Pagamento: Integração com gateway de pagamento para processar pagamentos via Pix e cartão.

## Style Guidelines:

- Primary color: Dark blue (#0D3B66) to inspire confidence and professionalism.
- Background color: Very light blue (#F1F1F1) for a neutral background that facilitates readability.
- Accent color: Green (#28A745) to signal success, completed payments, and availability.
- Headline font: 'Poppins' (sans-serif) for a geometric, precise and fashionable look. Body font: 'PT Sans' (sans-serif) for readability and a touch of warmth.
- Use flat, modern icons to represent sports, schedules, and payments.
- Implement a clean, responsive design that is easy to use on mobile devices, featuring a visual calendar for scheduling.
- Use subtle animations to provide feedback on user interactions, such as booking confirmations and payment status updates.
